# Lindex Miminko Výprodej Bot

## Nasazení na Railway

1. Vytvoř nový projekt na Railway a propojte s tímto GitHub repozitářem.
2. Přidejte Environment Variables:
   - TELEGRAM_TOKEN = <tvůj bot token>
   - CHAT_ID = <tvé chat_id>
3. Jako startovací příkaz nastavte:
   python lindex_bot_env.py
4. Nainstalujte Playwright:
   pip install -r requirements.txt
   playwright install
5. Deploy a bot poběží nonstop.
